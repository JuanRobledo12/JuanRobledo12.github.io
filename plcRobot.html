Human-Machine-Interface to Control a Cartesian Robot

The project was part of the final evaluation of the MR2019 Industrial Networks Project course at Tecnologico de Monterrey. The objective was to develop a human-machine interface (HMI) to manipulate a cartesian robot in a virtual machine. We used the automation software TIA Portal to generate the control software, the HMI, and the simulation. Before setting the HMI, it was necessary to design and implement the control software for the robot since it had two different operation modes.

The first operation mode is manual control, achieved with four buttons in the HMI, allowing the robot to move on the X, Y, and Z axes. Furthermore, users can manually activate a magnet in this mode to grab items from a sorting line. The second operation mode is the semi-automatic mode that allows users to press four buttons in the HMI to select which item the robot must collect. When a button is pressed, the robot will automatically move to the desired item, pick it, and move it to a storage area.

We have used ladder logic to develop the program since it was for a programmable logic controller (PLC). In addition, the program has other features such as counters to record the number of items the robot has processed. After developing the control software, we implemented the HMI and connected it with the software using the TIA Portal toolkit for HMI. The programmed PLC was the SIMATIC S7-300 from Siemens, and the HMI was the TP700 Comfort. Please check a small demo of the project in this YouTube video.
