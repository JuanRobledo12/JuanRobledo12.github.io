3DOF Teleoperated Planar Robotic Arm

I started working on this project in the course MR3031 Industrial Robotics at Tecnologico de Monterrey. The development involved building the robot and programming the control software.

I designed my robot using the CAD software SolidWorks and manufactured it with a fused deposition modeling (FDM) 3D printer. Then I calculated the forward and inverse kinematics equations that described the robot's motion. In addition, I developed a control software in Python where the user can input the desired coordinates to move the robot's end-effector. Moreover, the user can set the trajectory to be linear or angular.

Although it was not a requirement, I implemented an MQTT network to teleoperate the robot. First, in the Python control software, I created an MQTT client to publish the angle values that the arm's servomotors had to take to reach certain coordinates with a specific trajectory. Then I chose an ESP8266 to control the servomotors since it can be connected to WiFi and is compatible with the MQTT protocol. Therefore, I created a client in the servomotor's control software to subscribe to the topic where the angle values were being published. Finally, for the MQTT network to work, I used a Raspberry Pi and the Eclipse Mosquitto MQTT broker to handle the messages in my network.

The source code for this project is in my GitHub repository, "Karasu." If you are interested in more details about the project, please watch the following YouTube video.
